# mc-data-pack

Collection de data packs (et resource packs associés) pour Minecraft Java
Edition. Chaque sous-dossier est un projet indépendant avec son propre
README.

## Data packs

- [`ctf-aquatic/`](ctf-aquatic/README.md) — mini-jeu CTF générique avec items
  custom, pensé pour les maps aquatiques submergées.
